# Automated Ki-67 Proliferation Index Estimation
This repository contains the skeleton code for a two-stage deep learning pipeline for tissue and nuclei segmentation in histopathological images, as described in the Bachelor’s thesis on automated estimation of the Ki-67 proliferation index.

## Disclaimer
This repository contains the skeleton code developed for the Bachelor’s thesis. 
It **does not include any proprietary code** from CancerCenter.ai (which was used to download the datasets), nor the datasets used in the study. All datasets referenced in the thesis must be obtained separately.
